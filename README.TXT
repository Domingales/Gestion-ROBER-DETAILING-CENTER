ROBER DETAILING CENTER

Aplicación de gestión interna

Descripción

Aplicación web local (HTML + CSS + JavaScript) para la gestión diaria de ROBER DETAILING CENTER.
Permite controlar clientes, albaranes, facturas, gastos, stock y caja diaria, sin necesidad de conexión a internet ni servidor.

Los datos se almacenan localmente en el navegador.

Funcionalidades principales

Estado de la caja diaria

Facturas pendientes

Clientes registrados

Alertas de stock bajo

Caja diaria

Apertura y cierre de caja

Registro de movimientos de efectivo

Cálculo automático de descuadre

Clientes

Alta, edición y eliminación

Datos fiscales y de contacto

Albaranes

Registro de servicios realizados

Conversión posterior a factura

Facturas

Numeración automática

Control de cobro

Impresión / guardado en PDF

Gastos

Registro por proveedor y categoría

Stock

Productos y consumibles

Control de mínimos

Importar / Exportar

Exportación a CSV (Excel)

Importación desde CSV

Backup completo en JSON

Uso

Abrir el archivo index.html con Chrome o Edge
(recomendado abrir mediante servidor local si es posible).

Gestionar el negocio desde el menú lateral.

Realizar backup JSON periódicamente desde la barra superior.

Almacenamiento de datos

Los datos se guardan en el navegador mediante localStorage.

El almacenamiento tiene un límite aproximado de 5 MB.

Si se importan grandes volúmenes de datos, se recomienda:

Exportar backup

Limpiar la base local

O migrar a IndexedDB en futuras versiones

Copias de seguridad (recomendado)

Exportar backup JSON: botón superior

Restaurar backup JSON: botón superior

Recomendación: 1 copia semanal

Requisitos

Navegador moderno (Chrome, Edge, Firefox)
